# DevConnector

>Dev Connector Social Media for Developers

![DEVCONNECT](devconnector.png)

# Usage

Rename "config/production.json" to "config/default.json" and update value/settings to your own

## Install Dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

# Run in prod mode

npm start
```
